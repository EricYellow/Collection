# Perfect Utilities

除了core Web Service相关的功能, Perfect 提供了一系列基础构建块，用于构建服务器端和桌面应用程序。



就像之前章节中谈到的JSON library一样,  许多这样的功能可以通过Swift提供功能完成; 然而，Perfect的API旨在提高最终用户代码的效率，可读性和简单性。
